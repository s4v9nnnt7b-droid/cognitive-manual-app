# Codex / 実装担当AI向けプロンプト：認知書アプリ v0.1 Web MVP

あなたはNext.js / React / TypeScriptに強い実装エージェントです。
以下の方針で「認知書アプリ v0.1」のWebプロトタイプを実装・改善してください。

## 目的

心理検査結果や自己入力をもとに、利用者の認知特性を生活・勉強・対人関係で使える「認知書」「自分取扱説明書」「対人説明書」「勉強・仕事モード」「認知変化ログ」として整理する自己理解支援アプリを作成する。

このアプリは医療診断ではありません。
WAIS等の正式心理検査の実施・採点・診断を代替してはいけません。
出力は「傾向」「仮説」「負荷条件」として表現してください。

## 技術スタック

- Next.js App Router
- React
- TypeScript
- LocalStorage
- CSS ModulesまたはグローバルCSS
- AI APIはv0.1では使わない
- 本格OCRはv0.2以降
- 本格3Dキャラはv0.2以降

## 必須機能

1. オンボーディング同意
   - 医療診断ではない注意文
   - センシティブ情報に関する注意文
   - 同意チェック

2. プロフィール
   - ニックネーム
   - 利用目的
   - 心理検査の有無

3. Character OS Lite
   - 固定キャラ
   - 表情差分
   - 状態サマリー
   - 今日のひとこと

4. 心理テスト結果入力
   - 検査名
   - 検査日
   - FIQ / VIQ / PIQ / VC / PO / WM / PS
   - 当時の状態メモ
   - 0〜200以外は警告
   - すべて空欄では保存不可

5. 認知ロジック
   - 指数スコアを平均100、SD15の傾向として扱う
   - 差分を出す
   - PIQ−VIQ
   - PS−WM
   - 高速処理・視覚直感型などの仮判定

6. 出力
   - 心理テスト結果
   - 認知書
   - 自分取説
   - 対人説明書
   - 勉強・仕事モード
   - 認知変化ログ

7. 保存
   - LocalStorage保存
   - 全削除

## 禁止事項

- WAIS等の正式検査を代替すると表現する
- 医療診断として扱う
- 数値から障害名を断定する
- 能力の高低を人格評価に結びつける
- 個人の実名・医療情報・心理検査データをGitHubへ入れる

## 受け入れ条件

- `npm install` が通る
- `npm run dev` で起動できる
- 同意後にホームが表示される
- 入力値変更で判定と出力が更新される
- LocalStorage保存と全削除が動く
- v0.2候補、v0.3候補がUI上で区別されている
- READMEに非診断アプリであることが明記されている
