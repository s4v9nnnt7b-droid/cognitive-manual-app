# 認知書アプリ 決定事項監査メモ

## 確定事項

| 論点 | 決定 |
|---|---|
| アプリ名 | 仮称：認知書アプリ。別名候補：自分取扱説明書アプリ / Me Manual / Cognica / Self Manual OS |
| 目的 | 心理検査結果や自己入力を、生活・勉強・仕事・対人関係で使える認知書・自分取説へ変換する |
| v0.1本線 | 心理検査結果の手入力 |
| キャラ | Character OS Liteをv0.1から採用 |
| v0.1キャラ範囲 | 固定キャラ、表情5種、状態サマリー、今日のひとこと |
| 保存 | ローカル保存。クラウド同期は後回し |
| 安全 | 医療診断ではないこと、正式検査の代替ではないことを明示 |
| GitHub方針 | 実名・医療情報・個人の心理検査データは入れない |

## v0.1で作るもの

- オンボーディング同意
- プロフィール
- 心理テスト結果入力
- 認知タイプ仮判定
- 認知書
- 自分取説
- 対人説明書
- 勉強・仕事モード
- 認知変化ログ
- Character OS Lite
- LocalStorage保存
- 全削除

## v0.1で作らないもの

- 正式なWAIS等の検査実施
- 医療診断
- 治療方針の提示
- 本格OCR
- AI会話ログの自動取り込み
- 本格3Dアバター生成
- App Store公開前提の完全実装
- 課金導線
